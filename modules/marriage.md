## 婚配方位测算法（掌中诀）

### 核心原理

婚姻占卜需考虑多种因素：属相生合克害冲刑、六十花甲子纳音、命宫、八字、方位等。**婚配方位法**是其中一种重要的测算方法。

**重要提醒：** 婚配方位对的，不一定就是适合的伴侣，因为还有很多其他考虑因素。但可以规避错误选项。

### 掌中诀布局

伸出左手，用食指、中指、无名指和小指。

**从无名指指根开始，顺时针排列：**

```
无名指根部 → 子鼠
无名指指尖 → 丑牛
中指指尖 → 寅虎
食指指尖 → 卯兔
食指中部 → 辰龙
食指根部 → 巳蛇
手掌根部(小指侧) → 午马
小指根部 → 未羊
小指指尖 → 申猴
无名指指尖(另一侧) → 酉鸡
无名指中部 → 戌狗
无名指根部(另一侧) → 亥猪
```

**方位对应（上南下北）：**

| 地支 | 生肖 | 方位 | 说明 |
|------|------|------|------|
| 子 | 鼠 | 正北 | 四正之一 |
| 午 | 马 | 正南 | 四正之一 |
| 卯 | 兔 | 正东 | 四正之一 |
| 酉 | 鸡 | 正西 | 四正之一 |
| 其他地支 | - | 八方 | 四隅方位 |

**操作方法：** 将手掌中指指尖对准现实中的正南方，其他方位自然对应。

### 测算步骤

**第一步：确定月份起点**

阴历月份不是从子鼠开始，而是：
- **寅月 = 一月**
- 卯月 = 二月
- 辰月 = 三月
- 巳月 = 四月
- 午月 = 五月
- 未月 = 六月
- 申月 = 七月
- 酉月 = 八月
- 戌月 = 九月
- 亥月 = 十月
- **子月 = 十一月**
- **丑月 = 十二月**

**第二步：月上数日**

从月份对应的地支位置开始，顺时针数日期数。

**第三步：确定方位**

最终落点对应的地支方位，就是合适的伴侣方向（以出生地为准）。

### 实例演示

**案例：三月初三出生**

1. 三月 = 辰（龙）
2. 从辰位置开始数3天：辰→巳→午
3. 落在午（正南方）
4. **结论：** 南北方向是合适的伴侣方向

**案例：八月十五出生**

1. 八月 = 酉（鸡）
2. 从酉位置开始数15天：
   - 酉(1)→戌(2)→亥(3)→子(4)→丑(5)→寅(6)
   - 卯(7)→辰(8)→巳(9)→午(10)→未(11)→申(12)
   - 酉(13)→戌(14)→亥(15)
3. 落在亥（西北方）
4. **结论：** 西北方向是合适的伴侣方向

### 注意事项

1. **准确性验证：** 现实中有人符合，有人不符合，还有的一方符合另一方不符合
2. **综合判断：** 方位只是参考因素之一，还需结合：
   - 属相相合相克
   - 八字匹配度
   - 五行互补
   - 其他命理因素
3. **地理因素：** 以出生地为准，现代人口流动性大，需灵活理解

### 适用场景

- "我们两个生肖配不配？"
- "龙和兔能在一起吗？"
- "我们属相相冲怎么办？"

### 配对方法

只需要两个人的生肖（或出生年份）：

1. **三合配对**（大吉）：
   - 申子辰（猴鼠龙）- 水局
   - 寅午戌（虎马狗）- 火局
   - 巳酉丑（蛇鸡牛）- 金局
   - 亥卯未（猪兔羊）- 木局

- "详细分析一下我们的命盘匹配度"
- "我们适合结婚吗？"

### 合婚步骤

**前置信息收集：**

需要两个人的出生信息：
1. 你的出生日期、时间、性别
2. 对方的出生日期、时间、性别

**分析流程：**

参考 `references/bazi-matching.md`，按以下顺序分析：

1. **年柱配对**：
   - 天干合（甲己合、乙庚合、丙辛合、丁壬合、戊癸合）
   - 地支六合、三合、六冲、六害、三刑
   - 纳音五行生克

2. **日柱分析**：
   - 日主五行生克关系
   - 日支（配偶宫）的互动

3. **五行互补**：
   - 分析两人八字五行是否互补
   - 一方缺的五行，另一方是否充沛

4. **综合评分**

## 小六壬婚姻占卜

### 适用场景

- "我们什么时候能结婚?"
- "这段感情会有结果吗?"
- "我的姻缘什么时候到?"
- "我们两个合适吗?"

### 占卜方法

使用小六壬标准算法,针对感情问题进行解读。

### 六神感情断语

**大安(大吉):**
- 感情稳定,婚姻美满
- 单身者近期有良缘
- 已婚者家庭和睦
- **建议:** 把握机会,主动出击

**留连(平):**
- 感情进展缓慢
- 有拖延纠葛
- 暧昧不明
- **建议:** 耐心等待,多沟通

**速喜(吉):**
- 喜事将至
- 感情快速发展
- 良缘在即
- **建议:** 积极主动,把握时机

**赤口(凶):**
- 口舌是非
- 争执矛盾
- 第三者插足
- **建议:** 冷静处理,防范小人

**小吉(吉):**
- 小有波折但结果好
- 通过努力可成
- 需要时间磨合
- **建议:** 坚持不懈,真心相待

**空亡(大凶):**
- 感情不顺
- 有缘无分
- 难成正果
- **建议:** 重新审视,慎重考虑

### 双卦组合占卜

对双方分别占卜,组合解读:

| 组合 | 解读 |
|------|------|
| 大安+大安 | 天作之合,百年好合 |
| 大安+速喜 | 一见钟情,快速进展 |
| 大安+小吉 | 稳定发展,渐入佳境 |
| 速喜+速喜 | 热烈奔放,激情四射 |
| 小吉+小吉 | 平淡真挚,细水长流 |
| 留连+留连 | 纠缠不清,难分难解 |
| 赤口+赤口 | 争吵不断,矛盾重重 |
| 空亡+空亡 | 有缘无分,难以成全 |
| 大安+空亡 | 开始美好,结局遗憾 |
| 速喜+空亡 | 县花一现,难以持久 |

## 八字合婚分析

### 合婚要素

**1. 日柱配合**
- 日干相生相合为佳
- 日支相冲相克不利

**2. 五行互补**
- 一方缺水,一方水旺为佳
- 五行相生为佳,相克不利

**3. 命局配合**
- 男命财星旺,女命官星旺为佳
- 忌神互补为佳

**4. 大运同步**
- 大运走势一致为佳
- 一步运好一步运差需磨合

**5. 神煞配合**
- 桃花星不宜过多
- 红鸾天喜相逢为佳

### 合婚等级

**上等婚:**
- 五行完美互补
- 日柱天合地合
- 大运同步
- 无重大冲克

**中等婚:**
- 五行基本平衡
- 有冲克但可化解
- 大运部分同步
- 需要磨合

**下等婚:**
- 五行严重失衡
- 日柱相冲相克
- 大运背道而驰
- 矛盾难以调和

### 合婚示例

**男方:** 甲子年、丙寅月、戊辰日、壬子时
**女方:** 乙丑年、己卯月、庚午日、乙酉时

**分析:**
1. 日柱: 戊土(男)与庚金(女) - 土生金,男助女
2. 日支: 辰土与午火 - 火生土,女助男
3. 五行: 男缺金,女金旺 → 互补
4. 命局: 男财旺,女官旺 → 互相吸引
5. **结论:** 中上等婚配

**Q1: 我的姻缘什么时候到?**

占卜步骤:
1. 询问当前农历时间
2. 小六壬占卜
3. 根据六神判断时间

**时间推断:**
- 大安: 近期(1-3个月)
- 速喜: 快速(1个月内)
- 小吉: 中期(3-6个月)
- 留连: 延迟(半年以上)
- 赤口: 需化解阻碍
- 空亡: 暂时无望

**Q2: 我们能复合吗?**

占卜要点:
- 留连卦: 有机会,需时间
- 空亡卦: 难以复合
- 赤口卦: 矛盾未解
- 速喜卦: 快速复合

**Q3: 暗恋会有结果吗?**

占卜要点:
- 大安/速喜/小吉: 有希望
- 留连: 需要时间
- 赤口: 有竞争者
- 空亡: 单相思

## 实战案例

### 案例1: 何时有姻缘?

**用户:** 女,26岁,问何时能结婚
**时间:** 农历四月十八申时
**占卜:**
- 月: 大安→留连→速喜→赤口(四月)
- 日: 赤口起初一→小吉十八
- 时: 小吉起子时→空亡丑→大安寅→留连卯→速喜辰→赤口巳→小吉午→空亡未→大安申
- **结果:** 大安
- **解读:** 姻缘将至,近期(1-3个月)有良缘,应多参加社交活动

### 案例2: 能否复合?

**用户:** 男,问与前女友能否复合
**时间:** 农历七月廿三亥时
**占卜:**
- 月: 大安→留连→速喜→赤口→小吉→空亡→大安(七月)
- 日: 大安起初一→留连廿三
- 时: 留连起子时→速喜丑→赤口寅→小吉卯→空亡辰→大安巳→留连午→速喜未→赤口申→小吉酉→空亡戌→大安亥
- **结果:** 大安
- **解读:** 有复合机会,建议主动联系,真诚沟通
