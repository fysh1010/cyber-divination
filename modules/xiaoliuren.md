# 小六壬占卜算法

## 算法原理

小六壬是中国古代三大速占术之一,相传由诸葛亮改良流传至今。通过农历月份、日期、时辰三个时间参数,在手掌六宫位上顺时针推算,得出六神之一作为预测结果。

## 六神定义

### 掌诀位置

伸出左手，关注食指、中指、无名指，按以下精确位置划分：

**标准定位：**

```
食指根部 → 大安
食指指尖 → 留连
中指指尖 → 速喜
中指根部 → 空亡
无名指指尖 → 赤口
无名指根部 → 小吉
```

**顺时针循环顺序:**
大安 → 留连 → 速喜 → 赤口 → 小吉 → 空亡 → 大安

**详细说明：**
- 大安位于食指靠近手掌的指节（根部）
- 留连位于食指远离手掌的指节（指尖）
- 速喜位于中指远离手掌的指节（指尖）
- 空亡位于中指靠近手掌的指节（根部）
- 赤口位于无名指远离手掌的指节（指尖）
- 小吉位于无名指靠近手掌的指节（根部）

### 六神详解

| 六神 | 五行 | 方位 | 颜色 | 吉凶 | 主要含义 |
|------|------|------|------|------|----------|
| **大安** | 木 | 东方 | 青 | 大吉 | 静止、心安、吉祥、最大吉利 |
| **留连** | 水 | 北方 | 黑 | 平 | 暗昧不明、延迟、纠缠、拖延 |
| **速喜** | 火 | 南方 | 红 | 吉 | 快速、喜庆、吉利、时机已到 |

| **小吉** | 水 | - | - | 吉 | 和合、吉利、好结果将至 |
| **空亡** | 土 | 中央 | 黄 | 大凶 | 不吉、无结果、忧虑、最凶 |

## 计算步骤

### 标准算法(三月法)

**公式:** `(月 + 日 + 时) % 6`

**步骤:**
1. **定月份** - 从大安起正月,顺时针数到目标月份
2. **定日期** - 从月份落点起初一,顺时针数到目标日期
3. **定时报** - 从日期落点起子时,顺时针数到目标时辰

**示例:** 农历三月初五辰时

```
第一步: 月 → 大安(正月)→留连(二月)→速喜(三月)
第二步: 日 → 速喜起初一→空亡初二→大安初三→留连初四→赤口初五
第三步: 时 → 赤口起子时→小吉丑时→空亡寅时→大安卯时→留连辰时

结果: 留连
```

### 简化算法(快速计算法)

**公式:** `结果 = (月 + 日 + 时辰序号) % 6`

**时辰序号对照表:**
- 子(1), 丑(2), 寅(3), 卯(4), 辰(5), 巳(6)
- 午(7), 未(8), 申(9), 酉(10), 戌(11), 亥(12)

**计算规则:**
- 结果为0时,对应空亡
- 结果为1-5时,对应大安到小吉

**示例:** 农历三月初五辰时
```
月份: 3
日期: 5
时辰: 辰(5)
计算: (3 + 5 + 5) % 6 = 13 % 6 = 1

结果: 大安
```

**注意:** 图片示例使用了修正公式 `(月+日+时-2) % 6`,这是考虑到"月→日"和"日→时"的进位重叠。两种方法结果可能不同,建议使用标准算法(逐次推算)。

## 六神断语

### 大安
```
大安事事昌,求财在坤方,失物去不远,宅舍保安康
行人身未动,病者主无妨,将军回田野,仔细更推详
```
**解读:** 万事顺利,求财在西南,失物不远可寻,家人平安,出行无碍,病情无大碍。

### 留连
```
留连事难成,求谋日未明,官事凡宜缓,去者未回程
失物南方见,急讨方心称,更须防口舌,人口且平平
```
**解读:** 事情难成,拖延不决,官事宜缓,出行者未归,失物在南方,需防口舌之争。

### 速喜
```
速喜喜来临,求财向南行,失物申未午,逢人路上寻
官事有福德,病者无祸侵,田宅六畜吉,行人有信音
```
**解读:** 喜事将至,求财向南,失物可寻,官事有福,病情无碍,出行者有消息。

### 赤口
```
赤口主口舌,官非切宜防,失物速速讨,行人有惊慌
六畜多作怪,病者出西方,更须防咀咒,诚恐染瘟皇
```
**解读:** 口舌是非,官讼宜防,失物速寻,出行有惊,六畜不宁,病情需防,小心诅咒瘟疫。

### 小吉
```
小吉最吉昌,路上好商量,阴人来报喜,失物在坤方
行人即便至,交关甚是强,凡事皆和合,病者叩穷苍
```
**解读:** 吉祥顺利,路上遇贵人,有喜讯传来,失物在西南,出行者即归,凡事和谐,病情可愈。

### 空亡
```
空亡事不祥,阴人多乖张,求财无利益,行人有灾殃
失物寻一见,官事有刑伤,病人逢暗鬼,解禳保安康
```
**解读:** 诸事不祥,小人作祟,求财无果,出行有灾,失物难寻,官讼有伤,病情严重,需解禳化解。

## 应用场景

### 寻人寻物
根据六神方位和断语判断失物方向、能否找回

### 出行占卜
判断出行是否顺利、何时出发为宜

### 谈判签约
预测谈判结果、签约时机

### 求财问利
判断求财方向、时机是否成熟

### 婚姻感情
预测感情发展趋势

### 事业决策
辅助重大决策判断

## 注意事项

1. **占卜原则:**
   - 遇事即刻起卦,无事勿占
   - 凡事只占一次,再占不验
   - 同一事件只能用一种时间起卦

2. **时间要求:**
   - 使用农历时间
   - 建议使用真太阳时
   - 时间要精确到时辰

3. **心态要求:**
   - 保持敬畏之心,不可轻视
   - 占卜仅供参考,不可全信
   - 大方向正确即可,不必苛求细节

4. **准确性:**
   - 一般能预测准8成左右
   - 重大事件准确率更高
   - 日常小事仅供参考

## 代码实现示例

```python
def xiaoliuren_divination(month, day, hour):
    """
    小六壬占卜算法
    
    Args:
        month: 农历月份(1-12)
        day: 农历日期(1-30)
        hour: 时辰序号(1-12)
    
    Returns:
        六神名称和断语
    """
    liushen = ['空亡', '大安', '留连', '速喜', '赤口', '小吉']
    
    # 简化计算
    result = (month + day + hour) % 6
    
    return liushen[result]

# 标准算法(逐次推算)
def xiaoliuren_standard(month, day, hour):
    liushen = ['大安', '留连', '速喜', '赤口', '小吉', '空亡']
    
    # 第一步: 从大安起月
    month_result = (month - 1) % 6
    
    # 第二步: 从月份结果起日
    day_result = (month_result + day - 1) % 6
    
    # 第三步: 从日期结果起时
    final_result = (day_result + hour - 1) % 6
    
    return liushen[final_result]
```
