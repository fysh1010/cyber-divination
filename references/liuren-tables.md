## 六神详细断语

### 一、大安(大吉)

**口诀:**
```
大安事事昌,求财在坤方,失物去不远,宅舍保安康
行人身未动,病者主无妨,将军回田野,仔细更推详
```

**五行属性:** 木
**方位:** 东方
**颜色:** 青色
**吉凶:** 大吉
**主神:** 青龙

**传统占语:**
身不动时,五行属木,颜色青色,方位东方,临青龙,凡谋事主一、五、七。有静止、心安、吉祥之含义。

**详细解读:**

**事业:** 事业稳定发展,贵人相助,适合守成不宜冒进
**财运:** 求财在西南方,财运平稳,投资宜保守
**感情:** 感情稳定,婚姻美满,单身者有良缘
**健康:** 健康无碍,小病可愈,宜静养
**出行:** 出行平安,但不宜远行,宜在近处
**失物:** 失物不远可寻,在东南或西南方
**寻人:** 人在近处,未走远,可寻回
**官讼:** 官讼有利,宜和解,不宜争斗
**考试:** 考试顺利,成绩优异
**谋事:** 凡谋事主一、五、七,即初一、初五、初七或一、五、七月

**行动建议:**
- 保持现状,稳扎稳打
- 多休息,不宜过度操劳
- 把握机会,但不要冒进
- 注意细节,仔细推敲

### 二、留连(平)

**口诀:**
```
留连事难成,求谋日未明,官事凡宜缓,去者未回程
失物南方见,急讨方心称,更须防口舌,人口且平平
```

**五行属性:** 水
**方位:** 北方
**颜色:** 黑色
**吉凶:** 平
**主神:** 玄武

**传统占语:**
卒未归时,五行属水,颜色黑色,方位北方,临玄武,凡谋事主二、八、十。有暗昧不明、延迟、纠缠、拖延、漫长之含义。

**详细解读:**

**事业:** 事业进展缓慢,有阻碍,宜守不宜攻
**财运:** 求财不明朗,宜观望,不宜投资
**感情:** 感情纠葛,暧昧不明,需时间理清
**健康:** 健康平平,需注意调养,防慢性病
**出行:** 出行不利,有延迟,宜缓行
**失物:** 失物在南方,但需尽快寻找
**寻人:** 人未归,有拖延,需等待
**官讼:** 官讼宜缓,不宜速决,拖延有利
**考试:** 考试成绩一般,需努力
**谋事:** 凡谋事主二、八、十,即初二、初八、初十或二、八、十月

**行动建议:**
- 耐心等待,不要急于求成
- 多沟通,消除误会
- 防口舌是非,谨言慎行
- 调整心态,接受延迟

**详细解读:**

**财运:** 求财在南方,财运亨通,投资有利
**感情:** 喜事将至,感情快速发展,良缘在即
**健康:** 健康无碍,病情好转,心情愉快

**失物:** 失物在西南或南方,可寻回
**寻人:** 人即至,有消息传来
**官讼:** 官讼有福,有利解决
**考试:** 考试顺利,成绩优异
**谋事:** 凡谋事主三、六、九,即初三、初六、初九或三、六、九月

**行动建议:**
- 把握时机,积极行动
- 南方有利,宜向南发展
- 多参加社交活动
- 保持乐观心态
