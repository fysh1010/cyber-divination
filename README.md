## 核心算法

### 小六壬计算流程

```
步骤1: 定月份 - 从大安起正月,顺时针数到目标月
步骤2: 定日期 - 从月份落点起初一,数到目标日
步骤3: 定时辰 - 从日期落点起子时,数到目标时辰
```

**示例:** 农历三月初五辰时

```
月: 大安(1)→留连(2)→速喜(3)
日: 速喜起初一→...→赤口初五
时: 赤口起子时→...→留连辰时

结果: 留连(平) - 事难成,求谋日未明
```

### 八字排盘流程

1. 确定年柱(出生年份干支)
2. 确定月柱(月份干支)
3. 确定日柱(日期干支)
4. 确定时柱(时辰干支)
5. 分析日主强弱
6. 判断五行喜忌
7. 解读十神格局

## 注意事项

### 占卜原则

1. **遇事即刻起卦,无事勿占**
2. **凡事只占一次,再占不验**
3. **同一事件只能用一种时间起卦**
4. **保持敬畏之心,不可轻视术数**

### 时间要求

- 使用农历时间
- 时辰要准确(建议使用真太阳时)
- 时间精确到时辰
