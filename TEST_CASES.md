# 赛博算命 Skill 测试用例

## 测试场景 1: 小六壬占卜

**用户输入:** "帮我掐指一算,今天适合投资吗?农历三月初五辰时"

**预期流程:**
1. 识别为小六壬占卜需求
2. 提取参数: 月=3, 日=5, 时=辰(5)
3. 执行计算
4. 返回结果和解读

**计算过程:**
```
第一步: 月 → 大安(1月)→留连(2月)→速喜(3月)
第二步: 日 → 速喜起初一→空亡初二→大安初三→留连初四→赤口初五
第三步: 时 → 赤口起子时→小吉丑→空亡寅→大安卯→留连辰

结果: 留连
```

**预期输出:**
- 六神: 留连
- 吉凶: 平
- 解读: 事情难成,求谋日未明,投资宜观望
- 建议: 暂缓投资,等待时机

## 测试场景 2: 八字排盘

**用户输入:** "帮我算一下八字,男,1985年农历三月十五日寅时出生"

**预期流程:**
1. 识别为八字排盘需求
2. 收集性别、出生时间信息
3. 排出四柱
4. 分析日主强弱、五行喜忌
5. 给出事业、财运、感情建议

**预期四柱:**
```
年柱: 乙丑 (木土)
月柱: 庚辰 (金土)
日柱: 壬午 (水火)
时柱: 壬寅 (水木)
```

**预期分析:**
- 日主: 壬水,偏弱
- 喜用: 金、水
- 忌神: 土、火、木
- 格局: 食神格

## 测试场景 3: 婚姻占卜

**用户输入:** "我的姻缘什么时候到?女,26岁"

**预期流程:**
1. 识别为婚姻占卜
2. 询问当前农历时间
3. 执行小六壬占卜
4. 根据结果判断姻缘时间

**示例:** 农历四月十八申时
```
月: 赤口(4月)
日: 赤口起初一→小吉十八
时: 小吉起子时→...→大安申

结果: 大安
```

**预期输出:**
- 姻缘将至,近期(1-3个月)有良缘
- 建议: 多参加社交活动,把握机会

## 测试场景 4: 奇门遁甲

**用户输入:** "奇门遁甲排盘,看看今天的财运"

**预期流程:**
1. 识别为奇门遁甲需求
2. 确定当前时间
3. 排盘(简化版)
4. 重点分析生门落宫

**示例输出:**
- 时干落宫: xxx
- 生门(财门)落宫: 离九宫
- 分析: 生门临天英,火生土,财运亨通
- 建议: 南方求财,把握时机

## 测试场景 5: 综合对话

**用户:** "你好,我想算命"
**助手:** "您好!我可以为您提供多种占卜服务:
1. 小六壬 - 快速掐指占卜(适合日常小事)
2. 八字命理 - 四柱排盘分析
3. 婚姻占卜 - 感情姻缘预测
4. 奇门遁甲 - 高阶时空预测

请问您想了解哪方面呢?"

**用户:** "不知道,就是想看看今天的运势"
**助手:** "好的,我可以用小六壬为您占卜今日运势。请告诉我今天是农历几月几号,还有您想了解哪个时辰的运势?"

**用户:** "农历四月十八,现在下午3点"
**助手:** [执行计算]
"下午3点是申时。根据小六壬推算:

**结果: 大安**
**吉凶: 大吉**

**今日运势:**
- 整体运势顺畅,诸事吉祥
- 适合处理重要事务
- 贵人运旺,可寻求帮助
- 财运平稳,不宜大额投资
- 感情和谐,适合约会

**建议:**
把握今日好运势,处理重要决策。保持平和心态,避免冒进。"

## 错误处理测试

**场景1:** 用户输入不完整
**用户:** "帮我算一下"
**预期:** 提示用户选择占卜方式或提供更多信息

**场景2:** 时间格式错误
**用户:** "农历十三月初五" 
**预期:** 提示月份应在1-12之间

**场景3:** 时辰无法识别
**用户:** "下午13点是什么时辰"
**预期:** 自动转换为未时(13:00-15:00)
